# COVID-19 Data Analysis

## Overview
This project analyzes COVID-19 data using Python to identify trends in confirmed cases, recoveries, and deaths.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Features
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Data Visualization
- Trend Analysis
